## 🔨 Featured Projects

**[Food Waste Management](https://github.com/SNagarjuna07/Food-Waste-Management)**
Production-ready donation workflow backend. JWT + Google OAuth2, Redis-backed rate limiting, Kafka async notifications, MongoDB persistence, profile-based config, and Spring Boot Actuator for observability.

**[Finance Data Processing System](https://github.com/SNagarjuna07/Finance-Data-Processing-System)**
Role-based finance dashboard backend. Spring Boot, JWT auth, RBAC (VIEWER / ANALYST / ADMIN), paginated records with soft delete, and Swagger UI documentation.
